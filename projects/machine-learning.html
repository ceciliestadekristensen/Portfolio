About the project

This project was developed in collaboration with Bluestar PLM and explored how machine learning can support more efficient data entry in engineering-related systems. The goal was to identify patterns in historical data and generate context-aware suggestions for the next relevant input, reducing manual work and improving workflow efficiency.

Because the project was developed with an external company, the repository and this portfolio page focus on methodology, system structure and development approach rather than confidential production details.

My contribution

  • Worked on preprocessing and structuring the data for model input
  • Implemented label encoding for categorical variables
  • Contributed to the development of the machine learning pipeline
  • Worked with XGBoost integration and sequence-based recommendation logic
  • Helped visualize results and communicate the system value to external stakeholders

How the system worked

1
Data loading and filtering Historical data was prepared and filtered to create a usable input base for the model.
2
Preprocessing and label encoding Categorical values were transformed and structured to support machine learning-based prediction.
3
Feature transformation The data was shaped into a format suitable for model training and sequence-based logic.
4
Model training with XGBoost The model was trained to identify relevant patterns in prior selections and system behavior.
5
Prediction and recommendation The final system generated suggestions for the next relevant value based on previous inputs.
Due to NDA restrictions, the dataset and full implementation cannot be shared. This page focuses on methodology, pipeline design and project contribution rather than confidential data and company-specific details.

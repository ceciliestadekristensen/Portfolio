About the project

This project explored how AI systems can interpret and respond to human emotions in a more natural and engaging way. The solution combined a Unity-based interface with a Python backend and LLM-based response generation to create a real-time interaction flow between the user and the system. The project focused on both implementation and interaction design, showing how multiple technologies can be connected into one coherent user experience.

My contribution

  • Built the Unity-based interface and interaction flow
  • Worked on Python backend logic and system processing
  • Connected frontend and backend through API-based communication
  • Integrated LLM-based response generation
  • Designed and implemented emotionally aware interaction states
